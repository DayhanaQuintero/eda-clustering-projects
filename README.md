# EDA & Clustering Projects 📊

Collection of exploratory data analysis and clustering 
projects applied to real-world datasets.

## Projects

### 1. HR Analytics EDA
Exploratory analysis of employee data including 
null handling, duplicates, distributions and 
descriptive statistics.
- Dataset: general_data.csv (HR records)
- Tools: Python, pandas, seaborn, matplotlib

### 2. Credit Card Clustering
Customer segmentation using K-Means clustering 
on credit card usage data to identify behavioral 
patterns and spending profiles.
- Dataset: CC_GENERAL.csv
- Tools: Python, scikit-learn, seaborn, matplotlib
